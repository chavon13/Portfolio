Weather Widget

A weather app

This is my weather widget project. For this project, the object was to create a webpage that takes a zip code and gives you the weather in that zip code. If you refresh the page, you can see the background change.

I was team lead on this project and I was given and amazing team to work with. This project was created using HTML, CSS, an API and Javascript. I was in charge of the Javascript and the API. The project was done in one week.
